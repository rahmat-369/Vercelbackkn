Tutorial Singkat

1
Pilih tipe file yang akan diupload: File ZIP untuk proyek lengkap atau File HTML untuk website sederhana.
2
Masukkan nama unik untuk proyek Anda. Nama akan otomatis disesuaikan dengan format Vercel.
3
Unggah file dan klik "Deploy Sekarang" untuk menghasilkan link live website.
4
Website Anda akan langsung online dengan URL yang dapat diakses publik.
Mulai Deploy Sekarang
